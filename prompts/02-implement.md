# Phase 2 — Implementation for VietSalePro V7

You are working on **VietSalePro V7**, a modern Point of Sale & Inventory Management System.

## Task
Implement the next feature or improvement based on the analysis from Phase 1.

## Guidelines
1. Follow the existing project structure and conventions.
2. Use TypeScript strictly — no `any` types unless absolutely necessary.
3. Use Tailwind CSS v4 with the design tokens from `design-system-tokens.css`.
4. Reuse existing components from `components/`, `hooks/`, `lib/`, and `utils/`.
5. Use Supabase client from `services/` or `lib/` for database operations.
6. Add proper error handling and loading states.
7. Follow React best practices (hooks, memoization where needed).
8. Keep code clean, maintainable, and well-commented.

## Before Coding
- Review related existing files to understand patterns.
- Check `features.ts` and `constants.ts` for feature flags and constants.

## After Coding
- Run `npm run lint` (tsc --noEmit) to check TypeScript errors.
- Fix any type errors before finishing.
